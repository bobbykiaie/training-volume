# Training Volume Tracker

Training Volume Tracker is a React application for creating workout routines and tracking exercise sets. It allows users to visualize their daily workout volume, making it easier to manage and optimize their fitness routines.

This app also comes with an iOS version for convenience and portability.

[![App Screenshot](app-screenshot.png)](https://s11.gifyu.com/images/Simulator-Screen-Recording---iPhone-14-Pro---2023-05-11-at-16.49.54.gif)


## Features

- Create and save personalized workout routines
- Track the volume of exercise sets per muscle group
- Visualize daily workout volume with graphical charts
- User authentication system

## Tech Stack

- React
- Firebase for backend and authentication
- Material UI for component design

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

Make sure you have [Node.js](http://nodejs.org/) and [npm](https://www.npmjs.com/) installed.

### Installation

1. Clone the repository:

```bash
git clone https://github.com/yourusername/training-volume.git
```

2. Navigate into the cloned repository:

```bash
cd training-volume
```

3. Install the dependencies:

```bash
npm install
```

4. Run the application:

```bash
npm start
```

Then open [http://localhost:3000](http://localhost:3000) to view the app in your browser.
